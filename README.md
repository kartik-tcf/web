# MyWebsite

A modern, responsive website built with HTML, CSS, and JavaScript featuring a sleek dark theme with white text.

## Project Structure

```
web/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Stylesheet with dark theme
├── js/
│   └── script.js       # JavaScript for interactivity
└── README.md           # Project documentation
```

## Features

- 🎨 **Dark Theme**: Black background with white text and red accents
- 📱 **Responsive Design**: Mobile-friendly layout that works on all devices
- ✨ **Smooth Animations**: Scroll animations and hover effects
- 🧭 **Navigation**: Sticky navbar with smooth scrolling
- 💬 **Contact Form**: Interactive contact form with validation
- ⚡ **Performance**: Lightweight and fast-loading

## Sections

1. **Navigation Bar** - Sticky navbar with links to all sections
2. **Hero Section** - Eye-catching welcome area with call-to-action button
3. **About** - Brief description of the website
4. **Services** - Three service cards with hover effects
5. **Contact** - Contact form for user inquiries
6. **Footer** - Footer with copyright information

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No installation or dependencies required!

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #000;      /* Black background */
    --secondary-color: #fff;    /* White text */
    --accent-color: #ff6b6b;    /* Red accents */
    --dark-gray: #1a1a1a;
    --light-gray: #e0e0e0;
}
```

### Content
Simply edit the HTML content in `index.html` to customize:
- Site title and branding
- Navigation links
- Section content
- Service descriptions
- Contact form
